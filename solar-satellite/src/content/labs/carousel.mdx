### How it works
1. **Snap Scrolling:** We use CSS `snap-x` so it stops perfectly on images.
2. **Lazy Loading:** The images only load when they are near the viewport.
3. **No Frameworks:** This uses zero React/Vue. Just standard HTML/JS.
